# 🛍️ Online Shopping Home Page

This is a basic **Online Shopping Website Frontend** built using **HTML**, **CSS**, and **Bootstrap**. It provides a clean and responsive layout for displaying products with an "Add to Cart" option, a navigation header, and a footer.

---

## 🚀 Features

- ✅ Responsive layout using Bootstrap
- ✅ Multiple product sections
- ✅ "Add to Cart" buttons for each product
- ✅ Sticky header with navigation links
- ✅ Footer with useful information or links

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **Bootstrap 5**
